# Kevin Frantzen — Denver's Finest Milkman

A fan site for Kevin Frantzen, Denver's #1 door-to-door milkman, amateur rapper, and certified silly goose.

## Deploy to GitHub Pages

1. Create a new repo at github.com
2. Run:
   ```
   git remote add origin https://github.com/YOUR_USERNAME/kevin-frantzen-site.git
   git push -u origin main
   ```
3. Go to **Settings → Pages → Source → Deploy from branch → main / root**
4. Live at `https://YOUR_USERNAME.github.io/kevin-frantzen-site/`
