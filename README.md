## 🎮 Features

- Vintage editorial design — dark navy, cream & gold palette
- Animated falling milk drops in the hero section
- **WANTED** poster for The Mad Cow
- Fake testimonials from Denver neighbors
- **Mad Cow Hunt** — a playable top-down mobile game at the bottom of the page

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repo (e.g. `kevin-frantzen-site`)
2. Push these files to the `main` branch
3. Go to **Settings → Pages → Source → Deploy from branch → main / root**
4. Site will be live at `https://<your-username>.github.io/kevin-frantzen-site/`
